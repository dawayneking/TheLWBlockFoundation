Full Stack Development

Learn the basis of full stack and how it pertains to web development

User Experience/User Interface (UX/UI) Design

User Experience: Learn to make the interaction between users and services, such as an website, to retain more users.

User Interface: Learn to make the interface of a website user friendly, giving a complement and complimentary feel of product throughout interaction.

Responsive Web Design


Landing Page: While keeping User Experience and User Interface in mind, you can use principles to create landing and informational pages about your website. Hosting Website: Learn how to post your website, create whitepages for wikipedia, and manage github portfolio.

Front End Development

Learn how to use various framworks, create buttons, change fonts, and make the website interactive. Use web languages such as HTML and CSS to create your landing and informational pages. This will be your first steps in real development after creating initial webpage.

Back End Development and APIs

Learn how to really make your webpages interactive. Using various frameworks, and working on the server side of the website, learning developers focus on everything you can't see on a website. Ensure website functions, implement databases, Application Programming Interface (APIs), and architecture.

Database Applications

Learn how to use Database applications, and work with varius software to store or retrieve information. Create logins to authenticate users, and create structure that will allow for efficient storage and retrieval of data.

Machine Learning with Python

Work with open source frameworks that make neural networking applications easier to use. Create your portfolio with different simple startup opensource projects, using computer vision and tensor flow applications.
